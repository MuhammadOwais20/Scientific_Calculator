# Scientific Calculator in C

A standard calculator performs only basic functions and lacks advanced capabilities required in scientific and academic work. We aimed to develop a scientific calculator with extended functionality that supports operations such as logarithms, trigonometric calculations, derivatives, quadratic equations, and moreâ€”using structured and modular C programming.

---

## Features

- **Basic Arithmetic**: Addition, Subtraction, Multiplication, Division  
- **Powers and Roots**: Square, Cube, Power, Square Root  
- **Trigonometric Functions**: sin, cos, tan, sec, cosec, cot  
- **Logarithmic & Exponential Functions**  
- **Quadratic Equation Solver**  
- **Complex Number Operations**  
- **Derivatives**: Linear, Quadratic, and Cubic  
- **Percentage & GPA Calculator**  
- **Combination & Permutation**  
- **Unit Conversions**: Length, Mass, Time, Temperature, Force, Velocity, etc.  
- **Physics & Chemistry Constants**
